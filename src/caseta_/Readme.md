# Estructura de carpetas de caseta_
- Esta es la manera de como se va a trabajar en caseta
  
caseta_/
│
├── components/         # Componentes reutilizables (formularios, tablas, etc.)
│   └── FormularioCaseta.tsx
│
├── pages/              # Vistas principales (cada página de la app)
│   └── CasetaPage.tsx  # Ejemplo de página principal de caseta
│
├── services/           # Funciones para consumir la API (fetch, axios, etc.)
│   └── modulosService.ts
│
├── hooks/              # Custom hooks (si los necesitas)
│   └── useModulos.ts
│
├── utils/              # Funciones utilitarias (validaciones, helpers, etc.)
│
├── index.tsx           # Punto de entrada del módulo (puede ser para exportar todo)
│
└── styles/             # Archivos CSS/SCSS específicos del módulo
